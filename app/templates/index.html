Electricity Monitoring App

The webviewer allows you to view and create, modify and delete electricity readings and cost codes.

Cost codes are used to calculate the cost of electricity.

A plot is used to visualise the electricity use for each calendar year along with mean temperature data. The monthly mean temperature data at the Athenry weather station is obtained from the Met Eireann API: https://prodapi.metweb.ie/monthly-data/Athenry

The data may be viewed, created and modified in the webviewer:

Launch Webviewer

Using the API

To use the API to get json data, you can make the requests to the endpoints listed below:

    Endpoint Method Description
    /elec/units GET Get all electricity readings with associated cost code details
    /elec POST Create a new reading (JSON object containing 'year', 'month', 'units', 'cost_code')
    /elec/<int:id> PUT Update a reading (JSON object containing 'id', 'year', 'month', 'units', 'cost_code')
    /elec/<int:id> DELETE Delete a reading
    /elec/find GET Find a reading by year and month (Query parameters: 'year', 'month')
    /elec/cost_codes GET Get all cost codes
    /elec/cost_codes POST Create a new cost code (JSON object containing 'cost_code', 's_charge', 'unit_cost', 'vat_pc', 'supplier')
    /elec/cost_codes/<string:cost_code> PUT Update a cost code (JSON object containing 'id', 'cost_code', 's_charge', 'unit_cost', 'vat_pc', 'supplier')
    /elec/cost_codes/<string:cost_code> DELETE Delete a cost code
